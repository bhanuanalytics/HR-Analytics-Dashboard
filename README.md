# 📊 HR Analytics Dashboard

## 📌 Overview
This HR Analytics Dashboard is an interactive Power BI project designed to analyze employee data and uncover valuable insights into workforce performance, employee attrition, demographics, and job satisfaction. The dashboard helps HR professionals make data-driven decisions through dynamic visualizations and KPI tracking.

## 🎯 Project Objectives
- Analyze employee attrition trends.
- Monitor workforce demographics.
- Evaluate job satisfaction across different roles.
- Identify department-wise attrition patterns.
- Support employee retention and workforce planning strategies.

## 📈 Dashboard Features

### Key Metrics
- Total Employees: 1,470
- Active Employees: 1,233
- Attrition Count: 237
- Attrition Rate: 16.12%
- Average Age: 37 Years

### Visualizations
- Department-wise Attrition Analysis
- Employee Distribution by Age Group and Gender
- Job Satisfaction Rating Matrix
- Education Field-wise Attrition Analysis
- Attrition Count by Gender

### Interactive Elements
- Education Level Filters
- Dynamic KPI Cards
- Slicers for Customized Analysis
- Drill-down Visualizations

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Visualization

## 💡 Key Insights
- Research & Development recorded the highest attrition.
- Employees aged 26–35 represent the largest workforce segment.
- Job satisfaction varies significantly across roles.
- Life Sciences and Medical fields show higher attrition rates.
- Male employees contribute a larger share of total attrition.

## 📂 Dataset
The dataset contains employee demographic, educational, job role, and attrition-related information used to generate HR insights and workforce analytics.

## 🚀 Business Value
This dashboard enables organizations to:
- Improve employee retention strategies.
- Monitor workforce health and engagement.
- Identify high-risk attrition areas.
- Make informed HR decisions using data-driven insights.
